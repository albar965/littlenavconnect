Place a PDF manual "little-navconnect-user-manual-XX.pdf" with language code "XX" in
this directory to be used on computers that have this language setting.

An empty file with name "little-navconnect-user-manual-XX.online" indicates
that the language "XX" is also available online on GitBook.
This affects all help buttons and the links in the help menu which use the
online help on GitBook.
